# Lirik-showoff
Lirik lagu sederhana dengan animasi ketikan saat lagu diputar

compatible di segala macam mesin browser handphone anda (Android dan iPhone)

variable lrc : digunakan untuk memuat beberapa bait dari lirik lagu

variable speed : digunakan untuk gap/interval dari lirik sebelumnya dan selanjutnya

variable typedSpeed : digunakan untuk durasi saat lirik dinyanyikan

source code ini bersifat free lisence artinya kamu bisa mengambil, mengedit, menambahkan fungsi lain sesuai dengan kebutuhan.
